#This C programme used to swap two numbers without using third variable.

##Algorithm: \
###User input: \
    n1 = 5, n2 = 7; \

1. Method_1:
    n1 =  5 + 7 -> n1 = 12 \
    n2 = 12 - 7 -> n2 = 5  \
    n1 = 12 - 5 -> n1 = 7  \
    
2. Method_2:
    n1 =  5 * 7 -> n1 = 35 \
    n2 = 35 / 7 -> n2 = 5  \
    n1 = 35 / 5 -> n1 = 7 \
