#IsLeapYear:
    This C programme used to check the year is leap or non leap year.

##Criteria:
    
   1. if(year % 4 == 0) && (year % 100 != 0)        -> Leap year.

   2.   In case centurial year:
        if (year % 100 == 0) -> (year % 400 == 0)   -> Leap year.

    else                                            -> nonLeap Year.
