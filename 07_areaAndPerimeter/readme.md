#This C programme used to find the area and perimeter/circumference of basic shapes.    

##Formulae:                 
1. Circle:                  \
    area = pi * r * r;      \
    peri = 2 * pi * r;      

2. Rectangle:               \
    area = l * b;           \
    peri = 2 * (l+b);       

3. square:                  \
    area = s * s;           \
    peri = 4 * s;
